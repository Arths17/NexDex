## Installation

### macOS
1. Download `NexDex-Mac.zip`
2. Double-click to extract
3. Open `NexDex.app` (right-click ‚Üí Open if needed)
4. Dashboard opens automatically in your browser

### Windows (Coming Soon)
1. Download `NexDex-Windows.zip`
2. Right-click ‚Üí Extract All
3. Double-click `NexDex.exe`
4. Dashboard opens automatically

### Linux (Coming Soon)
1. Download `NexDex-Linux.tar.gz`
2. Extract: `tar xzf NexDex-Linux.tar.gz`
3. Run: `./NexDex/NexDex`
4. Dashboard opens automatically
